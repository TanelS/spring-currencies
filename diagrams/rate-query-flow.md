## GET /rates request flow

```mermaid
sequenceDiagram
    participant Client
    participant RateController
    participant GlobalExceptionHandler
    participant RateService
    participant CurrencyRepository
    participant RateRepository
    participant PostgreSQL

    Client->>RateController: GET /rates?baseCurrency=EUR&rateDate=27.06.2026&targetCurrency=USD,GBP

    Note over RateController: @PastOrPresent on rateDate<br/>@Validated on controller class<br/>→ rejects future dates (400)

    RateController->>RateService: getRatesforCurrencies(baseCurrency, rateDate, targetCurrency)

    RateService->>RateService: cleanString(baseCurrency).toUpperCase()
    RateService->>CurrencyRepository: findByCurrencyCode(baseCurrCleaned)

    alt Base currency not in DB
        CurrencyRepository-->>RateService: Empty Optional
        RateService->>GlobalExceptionHandler: throws CurrencyNotFoundException
        GlobalExceptionHandler-->>Client: 404 {"error": "Base currency 'XYZ' not found"}
    else Base currency found
        CurrencyRepository-->>RateService: Currency entity

        RateService->>CurrencyRepository: getAllCurrencyCodes()
        RateService->>RateRepository: findDates(localTimezone)

        RateService->>RateService: Normalise target codes to uppercase
        RateService->>RateService: Diff against DB codes → unknownCurrencies set

        alt rateDate is in findDates result
            RateService->>RateService: rateDatesToLoop = [rateDate]
        else rateDate is null or not found
            RateService->>RateService: rateDatesToLoop = all available dates
        end

        loop For each date in rateDatesToLoop
            RateService->>RateRepository: findRates(baseCurr, targetList, date, timezone)

            Note over PostgreSQL: DISTINCT ON (currency_code)<br/>(rate_date AT TIME ZONE tz)::date = :rateDate<br/>ORDER BY rate_date DESC → latest per day

            RateRepository-->>RateService: List of RateQueryResult (projection)

            alt queryResult empty AND baseCurrency in targetCurrency
                RateService->>RateService: rates = {baseCurrency: 1.0} (self-rate, no DB hit)
                RateService-->>RateController: return immediately
            else queryResult empty
                RateService->>RateService: continue (skip this date)
            else results found
                RateService->>RateService: Build rates HashMap from queryResult
                RateService->>RateService: Add BaseCurrencyRateDataOutput to list
            end
        end

        RateService-->>RateController: BaseCurrencyRateDataOutputList
        RateController-->>Client: 200 {"rates": [...], "unknownCurrencies": [...]}
    end
```

## Error responses summary

| Scenario | HTTP Status | Response |
|---|---|---|
| Invalid base currency | 404 | `{"error": "Base currency 'XYZ' not found"}` |
| Future date supplied | 400 | `{"error": "must be a date in the past or in the present"}` |
| Unknown target currencies | 200 | Valid rates + `"unknownCurrencies": ["XYZ"]` |
| No data for requested date | 200 | `{"rates": [], "unknownCurrencies": [...]}` |
| Self-rate (EUR/EUR) | 200 | `{"rates": [{"rates": {"EUR": 1.00}}]}` |
| Unexpected server error | 500 | `{"error": "Unexpected error: ..."}` |
